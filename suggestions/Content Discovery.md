# Content Discovery

## Related Prompts
- **Description**: Discover similar content automatically
- **Features**: Show prompts with similar tags or categories when viewing a prompt
- **Implementation**: Uses existing Supabase queries (free tier)
- **Benefit**: Helps users discover more relevant content

## Trending Prompts
- **Description**: Highlight popular content
- **Features**: Show most viewed or shared prompts from the current week
- **Implementation**: Uses Supabase queries (free tier)
- **Benefit**: Surfaces popular content and increases engagement

## Random Prompt Button
- **Description**: Discovery through serendipity
- **Features**: "Surprise me" button that shows a random prompt
- **Implementation**: Uses Supabase random query (free tier)
- **Benefit**: Fun way to discover new prompts and increase exploration
