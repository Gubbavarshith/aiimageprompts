# User Personalization

## User Preferences
- **Description**: Customizable default settings
- **Features**: Save user preferences like default category filter and items per page
- **Implementation**: Uses localStorage or Supabase (free)
- **Benefit**: Personalized experience for each user

## Prompt History
- **Description**: Track viewed content
- **Features**: Keep history of prompts user has viewed
- **Implementation**: Uses localStorage or Supabase (free tier)
- **Benefit**: Easy access to recently viewed prompts
