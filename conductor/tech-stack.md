# Tech Stack

## Frontend
- **Framework:** React 19.2.0
- **Build Tool:** Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Routing:** React Router DOM (v7)
- **State Management:** React Context (for Admin, Toast, etc.)
- **Animations:** Framer Motion
- **3D Rendering:** Three.js with @react-three/fiber and @react-three/drei
- **Charts:** Recharts (for Admin analytics)
- **Editor:** TipTap (Rich text editor for blog/prompts)

## Backend & Infrastructure
- **Authentication:** Clerk (@clerk/clerk-react)
- **Database & Storage:** Supabase (@supabase/supabase-js)
- **Email Service:** Resend
- **Deployment:** Vercel
- **Analytics:** Vercel Analytics & Speed Insights

## Development Tools
- **Linting:** ESLint with TypeScript ESLint plugin
- **Icons:** Lucide React, Heroicons
- **Utilities:** clsx, tailwind-merge, date-fns
- **UI Components:** Radix UI primitives
